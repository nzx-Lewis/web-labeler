## About

**WebLabeler** is a handy Google Chrome extension for web developers, testers, content managers, and others juggling multiple environments (development, testing, staging, and production) daily. With this lightweight extension, you can effortlessly create noticeable indicators that make your tabs easily distinguishable, helping you avoid mix-ups and costly mistakes.
